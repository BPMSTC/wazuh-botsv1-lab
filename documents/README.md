# Documents

This folder contains the primary instructor/operator documentation for the
Wazuh BOTSv1 classroom lab.

## Recommended reading order

1. `01-architecture.md`
   - End-to-end architecture, components, data flow, and design decisions.
2. `02-startup-and-operations.md`
   - Environment prerequisites, first-time startup, health validation,
     shutdown, and troubleshooting.
3. `03-class-demo-runbook.md`
   - Time-boxed classroom script with copy/paste commands, validation
     checkpoints, and fallback plans.
